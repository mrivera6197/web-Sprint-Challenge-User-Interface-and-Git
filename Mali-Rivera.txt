If you were to describe semantic HTML to the next cohort of students, what would you say?

semantic HTML is used to create meaning of the information on the web page. A semantic element describes its meaning in its name. For example  <header></header> creates the header of a webpage. It is best practice to use semantic HTML as much as possible.
